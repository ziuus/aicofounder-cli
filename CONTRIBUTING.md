## Contributing

Thanks for your interest in improving AICoFounder CLI.

### Before you start

- Open an issue describing the bug, feature, or UX improvement.
- Keep changes focused and easy to review.
- Make sure the CLI still runs locally after your change.

### Local setup

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

### Pull request guidelines

- Explain what changed and why.
- Include screenshots or terminal output for UI changes when helpful.
- Reference the related issue in the PR description.

### Code style

- Prefer small, readable changes.
- Match the existing Python style in the repository.
- Avoid unrelated refactors in the same PR.
